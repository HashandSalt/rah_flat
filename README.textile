h1. rah_flat

p. A plugin for "Textpattern CMS":http://www.textpattern.com. Edit template files, forms, pages as flat files. This plugin uses and experiments with the new template reading features included in Textpattern 4.6.0-dev.

h2. Requirements

p. Rah_flat at minimum requires:

* Textpattern 4.6.0-dev or newer.

h2. Installing

p. Using "Composer":http://getcomposer.org:

bc. $ composer.phar require rah/rah_flat
