h1. rah_flat

p. A plugin for "Textpattern CMS":http://www.textpattern.com. Edit template files, forms, pages as flat files. *Warning: this plugin will permanently remove your current templates when activated.*

h2. Requirements

p. Rah_flat at minimum requires:

* Textpattern 4.6.0-dev or newer.

h2. Installing

p. Using "Composer":http://getcomposer.org:

bc. $ composer require rah/rah_flat
